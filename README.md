## Installation

Clone the repository and install dependencies:

```bash
$ git clone https://github.com/DalcoveDev/INKINGIRESCURE_backend.git
$ cd INKINGIRescure_backend
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
